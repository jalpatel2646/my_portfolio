# 🚀 Jal Patel — Personal Portfolio

A modern, responsive personal portfolio website built with **React**, **Tailwind CSS**, and **Framer Motion** — featuring smooth animations, glassmorphism design, and a clean dark theme.

---

## ✨ Features

- 🎨 **Dark-themed UI** with glassmorphism & gradient accents
- 🌟 **Animated Loader** — smooth intro animation on page load
- 📱 **Fully Responsive** — works seamlessly on all screen sizes
- ⚡ **Framer Motion** — fluid page transitions and scroll animations
- 🧭 **Section Navigation** — smooth scrolling between all sections
- 💡 **Skills Grid** — organized by category with proficiency badges
- 🏆 **Projects & Hackathons** — live demo and GitHub links included
- 📜 **Certifications** — with clickable verification links
- 📬 **Contact Section** — clean contact form layout

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **React 18** | UI Framework |
| **Vite** | Build tool & dev server |
| **Tailwind CSS** | Utility-first styling |
| **Framer Motion** | Animations & transitions |
| **React Router DOM** | Client-side routing |
| **PostCSS + Autoprefixer** | CSS processing |

## 🚀 Getting Started

### Prerequisites

- **Node.js** v18+ installed
- **npm** v9+

### Installation

```bash
# Clone the repository
git clone https://github.com/jalpatel2646/my_portfolio.git

# Navigate into the project
cd my_portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at **http://localhost:5173**

### Build for Production

```bash
npm run build
```
